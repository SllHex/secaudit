# Screenshots

Place terminal screenshot PNG files here. Recommended captures:

| File | What to capture |
|------|----------------|
| `banner.png` | The matrix banner shown at scan start |
| `progress.png` | Live scanning progress table (mid-scan) |
| `complete.png` | `[ AUDIT COMPLETE ]` panel with score, grade, top issues |
| `interactive.png` | Control Deck splash + menu |
| `diff.png` | `[ DIFF ]` panel comparing two saved reports |
| `modules.png` | `secaudit modules` grouped overview |

**Recommended tool for capturing:** `terminalizer`, `asciinema`, or just a full-terminal window screenshot at 120+ columns with a dark theme.
